覚えるサービスまとめ

サービス名 説明
Dataproc Apach Spark / Hadoopのクラスタをフルマネージドで構築するサービス
Dataflow マネージドなApache Beamのランタイムを提供するサービス
Dataprep データの探索、クレンジングなどのパイプライン処理を、GUIで構築できるサービス
DataflowとBigQueryを基盤として構築される。
Cloud Data Fusion フルマネージドのデータパイプライン構築サービスであり、ノーコード・ローコードでETLを実装できる
Cloud Composer マネージドなApache Airflowを提供し、ワークフローの自動化とオーケストレーションをサポートするサービス
Dataplex データの管理、セキュリティ、ガバナンスを統合的に行うためのデータファブリックサービス
Data Catalog メタデータの管理やデータ資産の検索を支援するフルマネージドサービス
Datastream フルマネージドのデータレプリケーションサービスで、リアルタイムにデータを移行できる
Dataform SQLを活用したデータトランスフォーメーションを効率化するサービス
Workflows 分散アプリケーションやマイクロサービス間のワークフローを統合・オーケストレーションするサービス
Analytics Hub データの共有とコラボレーションを促進するBigQueryベースのプラットフォーム
BigQuery Omni 複数クラウド環境でのBigQueryクエリを可能にする分散分析サービス
BigLake データレイクとデータウェアハウスの統合管理を可能にするサービス
